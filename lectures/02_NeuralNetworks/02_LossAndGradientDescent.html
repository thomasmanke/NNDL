import numpy as np
import matplotlib.pyplot as plt

def sigmoid(z):
    return 1 / (1 + np.exp(-z))

# define truth (possibly simple)
def y_true_fn(theta, x):
    y_true = ( theta[0] + 
        theta[1]*np.sin(theta[2] * x + theta[3]) +
        theta[4]*np.cos(theta[4] * x - theta[6]) +
        theta[7]*np.sin(theta[8] * x + theta[9])
    )
    # complexity of true model is irrelevant for shape of loss surface
    y_true = theta[0] + theta[1]*x 

#   optional: turn into binary
#    y_true = (y_true > np.median(y_true)).astype(int) 
    return y_true

# define model predictions (complex)
def y_pred_fn(w1, w2, x):
#    y_pred = sigmoid(w1 + w2 * x)
    y_pred = np.sin(np.pi*x + np.sin(w1*np.pi*x)) * np.cos(w2*x)
    return y_pred

def MSEloss(w1, w2):
    y_pred = y_pred_fn(w1, w2, x[:, None, None])
    return np.mean((y_true[:, None, None] - y_pred)**2, axis=0)

def CEloss(w1, w2):
    y_pred = y_pred_fn(w1, w2, x[:, None, None])
    eps = 1e-12 # avoid log(0)
    y_pred = np.clip(y_pred, eps, 1 - eps)
    # binary cross entropy
    bce = -(y_true[:, None, None] * np.log(y_pred) +
            (1 - y_true[:, None, None]) * np.log(1 - y_pred))
    return np.mean(bce, axis=0)

n = 240
x = np.linspace(-np.pi, np.pi, n)
theta = [9, 1.2, 3.0, 0.3, -0.8, 5.0, -0.1, 0.6, 7.0, 0.7, 0.2]
y_true = y_true_fn(theta,x)

#plt.figure(figsize=(6, 5))
#plt.scatter(x,y_true)
#plt.show()

# construct 2D parameter grid
w1 = np.linspace(-6, 6, 200)
w2 = np.linspace(-6, 6, 200)
W1, W2 = np.meshgrid(w1, w2)

Z = MSEloss(W1, W2)

plt.figure(figsize=(6,5))
plt.contourf(W1, W2, Z, levels=100, cmap='viridis')
plt.colorbar(label='Loss')
plt.xlabel('w1')
plt.ylabel('w2')
plt.title('Non-convex Loss Surface')
plt.show()

import numpy as np
import matplotlib.pyplot as plt

# Define the loss function and its derivative (gradient)
def L(w):
    return (w - 2)**2 + 1   # Parabola with minimum at w=2

def dL(w):
    return 2 * (w - 2)

# Choose a point away from the minimum
w0 = 0.5
L0 = L(w0)
grad = dL(w0)

# Define tangent line at w0
def tangent(w):
    return L0 + grad * (w - w0)

# Create range for plotting
w = np.linspace(-1, 3, 200)

# Plot
plt.figure(figsize=(6, 6))
plt.plot(w, L(w), label='Loss $L(w)$', color='blue')
plt.plot(w, tangent(w), '--', color='red', label='Tangent at $w_0$')
plt.scatter([w0], [L0], color='red', zorder=5, label=f'initial $w_0 = {w0}$')
plt.scatter([2], [L(2)], color='green', zorder=5, label='Minimum')

# Annotate gradient direction
plt.arrow(w0, L0, 0.6, grad*0.6, head_width=0.1, head_length=0.3, fc='red', ec='red')

# Labels and style
plt.xlabel('Weight $w$')
plt.ylabel('Loss $L(w)$')
plt.title('Simple Loss Function')
plt.legend()
plt.grid(True)
plt.tight_layout()
plt.show()
Gradient Descent
  • changes in parameter \(w\) will propagates through network to affect \(L\)
  • Parameter Update \[ w = w - \alpha \frac{dL}{dw} \]
  • similar for bias term \(b\)
  • local derivative: \(\frac{dL}{dw} = f(w)\)
  • vectorization: \(w\) and \(\frac{dL}{dw}\) are vectors of same dimension. Derivatives \(\to\) Gradients
  • \(\alpha\): learning rate (hyperparameter)

Important Derivatives

How to calculate local slopes

Function \(f(x)\) Derivative \(f'(x)=\frac{df}{dx}\)
\(5x\) \(5\)
\(5x^2\) \(10x\)
\(e^x\) \(e^x\)
\(e^{-x}\) \(-e^{-x}\)
Sigmoid: \(\frac{1}{1 + e^{-x}}\) \(\frac{e^{-x}}{(1 + e^{-x})^2} = f(x)(1 - f(x))\)
\(\tanh(x)\) \(1 - f(x)^2\)
ReLU: \(max(0, x)\) \((0,1)\)
\(f(g(x))\) \(f'(g(x)) \cdot g'(x) \longrightarrow\) chain rule

Gradient Descent

each sample \(i\) contributes to overall loss and overall gradient \[ \begin{aligned} L(w) &= \frac{1}{N}\sum_i L_i(w) \\ \frac{dL}{dw} &= \frac{1}{N}\sum_i \frac{dL_i}{dw} \\ w &= w - \alpha \frac{d L}{dw} \end{aligned} \]

  • Initialization (random): \(w_0\)
  • step-wise update is expensive (sum over all data)
  • in practice this is done for **batches* of training samples

Forward Calculation

Example: Logistic regression

The loss function (binary cross entropy): \[ L(a,y) = -y \log(a) - (1-y)\log(1-a) \]

Minimal Loss = Maximal Likelihood
  • observed data: \(x\)
  • true label: \(y\)
  • predicted label probability: \(p =f(x, \theta)\)

Conditional Probability: \[ Pr(\mbox{Label}=y|x, \theta) = p^y (1-p)^{(1-y)} \to \mbox{maximize !} \]

Loss for one sample \(x\): \[ L(p, y) = - \log Pr(y|x, \theta) = - y \log p - (1-y) \log (1 - p) \to \mbox{minimize !} \]

Loss for all \(N\) samples \((x_1,y_1), (x_2,y_2) \ldots (x_N, y_N)\) and predicted probabilities \(p_i = f(x_i, \theta)\): \[ L(x,y) = \frac{1}{N} \sum_i L(p_i, y_i) \]

Computational Graph

Backpropagation

Chain Rule

\[ \begin{aligned} \frac{\partial L}{\partial w} &= \left(\frac{\partial L}{\partial a}\right)\left(\frac{\partial a}{\partial z}\right) \left(\frac{\partial z}{\partial w}\right) \end{aligned} \]

\[ \begin{aligned} \frac{\partial L}{\partial a} &= -\frac{y}{a} + \frac{1-y}{1-a}\\ \frac{\partial a}{\partial z} &= a(1-a)~~~\mbox{for sigmoid}\\ \frac{\partial z}{\partial w_j} &= x_j~~~; ~~~\frac{\partial z}{\partial b} = 1 \end{aligned} \]

Backward Calculation
  • derivatives = lookup and multiplication
  • decompose larger calculation into smaller local parts
  • strict computational flow ensures that all required contributions are known
  • importance of differentiable activation functions: \(\frac{\partial a}{\partial z}\)
  • attaches a gradient to each forward value: \(a \to da \equiv \frac{\partial L}{\partial a}\)
  • flexibility: other loss functions, other inputs, more layers
  • this is for one sample \((\mathbf{x},y)\) only

Summary: Forward and Backward Flows

The simple example (1 layer logistic regression) generalizes

  • to deeper neural network with multiple layers
  • parameters (weights and biases) at each layer \(l\)
  • activities \(a\) are calculated forward
  • gradients \(dL/da \equiv da\) are calculated backward
Important
  • gradients are byproduct of backpropagation
  • shapes of gradients agree with shape of tensors: \([\frac{\partial L}{\partial a}] = [da] = [a]\)
  • matrix notation for book-keeping/simplification and vectorized speedup
  • lookup and matrix multiplication

## Gradient Descent

each sample $i$ contributes to overall loss and overall gradient
$$
\begin{aligned}
L(w) &= \frac{1}{N}\sum_i L_i(w) \\
\frac{dL}{dw} &= \frac{1}{N}\sum_i \frac{dL_i}{dw} \\
w &= w - \alpha \frac{d L}{dw}
\end{aligned}
$$

- Initialization (random): $w_0$
- step-wise update is expensive (sum over all data)
- in practice this is done for **batches* of training samples

## Forward Calculation

$$
L(a,y) = -y \log(a) - (1-y)\log(1-a)

$$

## Backpropagation


::: {.callout-important}
### Chain Rule

$$
\begin{aligned}
\frac{\partial L}{\partial w} &= \left(\frac{\partial L}{\partial a}\right)\left(\frac{\partial a}{\partial z}\right) \left(\frac{\partial z}{\partial w}\right)
\end{aligned}
$$
:::


$$
\begin{aligned}
\frac{\partial L}{\partial a} &= -\frac{y}{a} + \frac{1-y}{1-a}\\ 
\frac{\partial a}{\partial z} &= a(1-a)~~~\mbox{for sigmoid}\\ 
\frac{\partial z}{\partial w_j} &= x_j~~~; ~~~\frac{\partial z}{\partial b} = 1 
\end{aligned}
$$

::: callout-note
### Backward Calculation
- derivatives = **lookup** and **multiplication**
- decompose larger calculation into smaller local parts
- strict computational flow ensures that all required contributions are known
- importance of **differentiable** activation functions: $\frac{\partial a}{\partial z}$
- attaches a gradient to each forward value: $a \to da \equiv \frac{\partial L}{\partial a}$
- flexibility: other loss functions, other inputs, more layers
- this is for one sample $(\mathbf{x},y)$ only
:::

## Summary: Forward and Backward Flows

The simple example (1 layer logistic regression) generalizes 

- to deeper neural network with multiple layers
- parameters (weights and biases) at each layer $l$
- activities $a$ are calculated forward
- gradients $dL/da \equiv da$ are calculated backward

::: callout-important
- gradients are byproduct of backpropagation
- shapes of gradients agree with shape of tensors: $[\frac{\partial L}{\partial a}] = [da] = [a]$
- matrix notation for book-keeping/simplification and vectorized speedup
- lookup and matrix multiplication
:::
