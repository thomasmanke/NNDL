Inserting non-linearities in neural networks
Author

Thomas Manke

Published

November 14, 2025

Softmax

… for multinomial classification with \(K\) classes (in output layer)

\[ z_k = w_k \cdot x + b_k \longrightarrow p_k = \frac{e^{z_k}}{\sum_{l=0}^{K-1} e^{z_l}}~~~~k \in [0, ..., K-1] \]

  • shift invariance (for probabilities):
    \(p_k = f(z_k) = f(z_k + c)\)
  • scale invariance (for class predictions): \(\hat y_k = \arg \max_k f(z_k) = \arg \max_k f(z_k/T) =\arg \max_k z_k\)
Code 
import torch
import matplotlib.pyplot as plt
import torch.nn.functional as F
from itertools import product

z = torch.tensor([-0.35, 1.5])
C = [0.0, 33.3, 0.0, 0.0]
T = [1.0, 1.0, 5.0, 0.50]
labels = [f"c={c}, T={t}" for c, t in zip(C,T)]
probs = torch.stack([F.softmax((z + c)/t, dim=0) for c, t in zip(C,T)])
x = torch.arange(len(labels)).numpy()
p0 = probs[:, 0].numpy()
p1 = probs[:, 1].numpy()

# --- stacked barplot ---
plt.figure(figsize=(6, 4))
plt.bar(x, p0, label="class 0", color="steelblue")
plt.bar(x, p1, bottom=p0, label="class 1", color="orange")
plt.axhline(0.5, color="gray", linestyle="--", linewidth=1)

plt.xticks(x, labels, rotation=30, ha="right")
plt.ylim(0, 1)
plt.ylabel("Probability")
plt.title("Softmax probabilities (with Shifts and Scales)")
plt.legend(loc="upper right")
plt.tight_layout()
plt.show()
Note
  • interprete output of NN as probabilities with \(\sum p_k = 1\)
  • softmax uses all neurons (of output layer)
  • no hyper-parameter

Log-sum Trick

How to avoid overflow \(e^{z_k}\) for large \(|z_k|\) ?

Define maximal \(z_k\)

\[ \begin{aligned} z^\ast &= \max_k z_k \\ \log\sum_l e^{z_l} &= z^\ast + \log \sum_l e^{z_l - z^\ast} \end{aligned} \]

This trick is used by F.softmax(z) and other professional implementations of softmax.

Sigmoid

… a special case for two output variables: binary logistic regression

\[ \left[z_0, z_1\right] \to \left[ p_0 = \frac{e^{z_0}}{e^{z_0} + e^{z_1}}, p_1 = 1 - p_0 \right] \\ \]

Binary Classification can be realized with single output neuron

\[ p_0 = \sigma(z_0) = \frac{1}{1+\exp(-z_0)} \]

Binary Cross Entropy
  • special case of Cross Entropy 2 classes (binary output).

Loss for a single sample \((x,y) \to (z, y) \to (p, y) \to (\hat y, y)\)

\[ L(p, y) = - \sum_l y_l \log p_l = - y_0 \log p_0 - (1-y_0) \log( 1 - p_0) \]

True class label: \(y_0 \in \{0, 1\}\) and \(y_1 = 1 - y_0\)

Total loss = average loss for all samples \((x_i,y_i)\)

\[ L = \frac{1}{N} \sum_i L(p_i, y_i) \]

Code 
z = torch.linspace(-5, 5, steps=200)
y_sigmoid = torch.sigmoid(z)
plt.figure(figsize=(8, 5))
plt.plot(z, y_sigmoid, label='Sigmoid', linewidth=2)
plt.title("Sigmoid Function")
plt.xlabel("z")
plt.ylabel("σ(z)")
plt.grid(True) 
plt.show()
Note
  • input: \(-\infty \ldots +\infty\)
  • output: \(0 \ldots 1\) (probability interpretation)
  • sharp transitions, position determined by bias
  • narrow range of changes & asymptotically flat
  • sigmoid can also be used for internal neurons
  • sigmoid works independently for individual neurons
  • no additional hyper-parameter

Other Activations Functions

Introduce non-linearities also for internal layers

2-layer network without non-linearities

\[ \begin{aligned} z_1 & = W_1 \cdot x + b_1 \\ z_2 & = W_2 \cdot z_1 + b_2 \\ & = W_2 ( W_1 \cdot x + b_1) + b_2 \\ &= W^\prime x + b^\prime \end{aligned} \]

… useless and corresponds to 1-layer network !

Use (non-linear) activation functions \(a(z)\)

\[ \begin{aligned} z_1 & = W_1 \cdot x + b_1 \\ a_1 & = a(z_1) \\ z_2 & = W_2 \cdot a_1 + b_2 \\ a_2 & = a(z_2) \end{aligned} \]

  • many activation functions \(a(z)\) are possible
  • should be simple and differentiable

tanh

\[ \begin{aligned} \tanh(z) &= \frac{e^z - e^{-z}}{e^z + e^{-z}} = 2 \sigma(2z) - 1 \end{aligned} \]

Code 
y_tanh = torch.tanh(z)
plt.figure(figsize=(8, 5))
plt.plot(z, y_tanh, label='tanh', linewidth=2)
plt.title("tanh")
plt.xlabel("z")
plt.ylabel("tanh(z)")
plt.grid(True)
plt.show()
Note
  • input: \(-\infty \ldots +\infty\)
  • output: \(-1 \ldots 1\)
  • “centered” and scaled sigmoid
  • sharp transitions, position determined by bias
  • small range of changes & asymptotically flat
  • zero-centered: positive and negative output

ReLU: Rectified Linear Unit

\[ ReLU(z) = \max(0,z) \]

Code 
y_relu = F.relu(z)
plt.figure(figsize=(8, 5))
plt.plot(z, y_relu, label='ReLU', linewidth=2)
plt.title("ReLU")
plt.xlabel("z")
plt.ylabel("ReLU(z)")
plt.grid(True)
plt.show()
Note
  • input: \(-\infty \ldots +\infty\)
  • output: \(0 \ldots \infty\)
  • very simple and cheap
  • no saturation for \(x>0\)
  • only discontinuity at \(x=0\)

… and many more activation functions possible.

## Softmax

... for multinomial classification with $K$ classes (in output layer)

$$
z_k = w_k \cdot x + b_k \longrightarrow p_k = \frac{e^{z_k}}{\sum_{l=0}^{K-1} e^{z_l}}~~~~k \in [0, ..., K-1]
$$


- shift invariance (for *probabilities*):           
        $p_k = f(z_k) = f(z_k + c)$
- scale invariance (for *class predictions*): 
        $\hat y_k = \arg \max_k f(z_k) = \arg \max_k f(z_k/T) =\arg \max_k z_k$

```{python}
#| label: softmax_behaviour
#| code-fold: true
#| fig-cap: Shift invariance of probabilities, Scale invariance of predictions, Confidence and Over-Confidence. Here for 2 categories with z = [-0.35, +1.5].
import torch
import matplotlib.pyplot as plt
import torch.nn.functional as F
from itertools import product

z = torch.tensor([-0.35, 1.5])

p1 = probs[:, 1].numpy()

plt.bar(x, p0, label="class 0", color="steelblue")
plt.bar(x, p1, bottom=p0, label="class 1", color="orange")
plt.axhline(0.5, color="gray", linestyle="--", linewidth=1)

plt.xticks(x, labels, rotation=30, ha="right")
plt.ylim(0, 1)

plt.show()
```

$$
\begin{aligned}
z^\ast &= \max_k z_k \\
\log\sum_l e^{z_l} &= z^\ast + \log \sum_l e^{z_l - z^\ast}
\end{aligned}

## Sigmoid

... a special case for two output variables: binary logistic regression

$$
\left[z_0, z_1\right] \to \left[ p_0 = \frac{e^{z_0}}{e^{z_0} + e^{z_1}}, p_1 = 1 - p_0 \right] \\
$$

Binary Classification can be realized with **single** output neuron

$$
p_0 = \sigma(z_0) = \frac{1}{1+\exp(-z_0)}
$$


::: {.callout-note}
### Binary Cross Entropy

- special case of Cross Entropy 2 classes (*binary output*).

Loss for a single sample $(x,y) \to (z, y) \to (p, y) \to (\hat y, y)$

$$
L(p, y) = - \sum_l y_l \log p_l = - y_0 \log p_0 - (1-y_0) \log( 1 - p_0)
$$

True class label: $y_0 \in \{0, 1\}$ and  $y_1 = 1 - y_0$

Total loss = average loss for all samples $(x_i,y_i)$

$$
L = \frac{1}{N} \sum_i L(p_i, y_i)
$$

:::


```{python}
#| label: sigmoid
#| code-fold: true
z = torch.linspace(-5, 5, steps=200)
y_sigmoid = torch.sigmoid(z)
plt.figure(figsize=(8, 5))
plt.plot(z, y_sigmoid, label='Sigmoid', linewidth=2)
plt.title("Sigmoid Function")
plt.xlabel("z")
plt.ylabel("σ(z)")
plt.grid(True) 
plt.show()
```

::: {.callout-note}
- input: $-\infty \ldots +\infty$
- output: $0 \ldots 1$ (probability interpretation)
- sharp transitions, position determined by *bias*
- narrow range of changes & asymptotically flat
- sigmoid can also be used for internal neurons
- sigmoid works independently for individual neurons
- no additional hyper-parameter
:::



## Other Activations Functions

Introduce non-linearities also for **internal layers**

2-layer network *without* non-linearities

$$
\begin{aligned}
z_1 & = W_1 \cdot x + b_1 \\
z_2 & = W_2 \cdot z_1 + b_2 \\
 & = W_2 ( W_1 \cdot x + b_1) + b_2 \\
 &= W^\prime x + b^\prime
\end{aligned}
$$

... useless and corresponds to 1-layer network !


::: callout-tip
### Use (non-linear) activation functions $a(z)$

$$
\begin{aligned}
z_1 & = W_1 \cdot x + b_1 \\
a_1 & = a(z_1) \\
z_2 & = W_2 \cdot a_1 + b_2 \\
a_2 & = a(z_2)
\end{aligned}
$$

- many activation functions $a(z)$ are possible
- should be simple and differentiable
:::

### tanh 
$$
\begin{aligned}
\tanh(z) &= \frac{e^z - e^{-z}}{e^z + e^{-z}} = 2 \sigma(2z) - 1
\end{aligned}
$$

```{python}
#| label: tanh
#| code-fold: true
y_tanh = torch.tanh(z)
plt.figure(figsize=(8, 5))
plt.plot(z, y_tanh, label='tanh', linewidth=2)
plt.title("tanh")
plt.xlabel("z")
plt.ylabel("tanh(z)")
plt.grid(True)
plt.show()
```

::: {.callout-note}
- input: $-\infty \ldots +\infty$
- output: $-1 \ldots 1$
- "centered" and scaled sigmoid
- sharp transitions, position determined by *bias*
- small range of changes & asymptotically flat
- *zero-centered*: positive and negative output
:::

### ReLU: Rectified Linear Unit

$$
ReLU(z) = \max(0,z)
$$
