Data Rates

Domain Open Formats Proprietary Formats Typical Rates
Audio FLAC, WAV (MP3), M4A (Apple), WMA (Microsoft) 1–10 MB/min
Video WebM/VP9 (MP4/H.264), MOV (Apple), WMV (Microsoft) 10 MB/min (HD) → GBs/min (4K+)
Internet (Common Crawl) WARC (HTTP) - 100 TB/ month ~ 2GB/min
Genomics (sequencing machines) FASTQ, VCF, BAM, … BCL (Illumina), pod5 (ONT) 34 GB/min → 2 GB/min
Earth Observation System (NASA) HDF5 - ?? → 2 GB/min
Particle physics (CERN LHC) ROOT - 1 PB/s → 2000 GB/min
Square Kilometer Array HDF5, FITS - 75 TB/min → 1 TB/min
Note
  • on-the-fly processing for large data
  • ship metadata with data
  • avoid data duplication
  • compression

Data Analysis

Data Analysis \(\gg\) Modeling
Stage Example tasks Typical time share
Data Acquisition gathering data and metadata, checking consistency, negotiating access 10–20 %
Data Cleaning Handling missing values, reshaping tables, fixing types, deduplication 20–60 %
Data Transformation Normalization, scaling, filtering, creating derived variables 20–30 %
Data Exploration Visualization, summary statistics, feature selection, sanity checks 10–20 %
Data Modeling Model selection, Model training/(deep)learning, evaluation 5–20 %
Data Communication Reports, dashboards, storytelling 5–10 %
Note

Most of this course will focus on “Data Modeling”

### image data

#| label: read_image
from PIL import Image
import numpy as np
import requests
from io import BytesIO
import matplotlib.pyplot as plt

url = 'https://img-datasets.s3.amazonaws.com/elephant.jpg'
response = requests.get(url)
fo = BytesIO(response.content) # content2fileobject
img_obj = Image.open(fo)       # image object

| ----------------------- | ---------------------------------------------------------------------- | ------------------ |
| **Data Acquisition**    | gathering data and metadata, checking consistency, negotiating access  | **10–20 %**        |

| **Data Transformation** | Normalization, scaling, filtering, creating derived variables          | **20–30 %**        |
| **Data Exploration**    | Visualization, summary statistics, feature selection, sanity checks    | **10–20 %**        |
| **Data Modeling**       | Model selection, Model training/(deep)learning, evaluation     | **5–20 %**        |
| **Data Communication**  | Reports, dashboards, storytelling                                      | **5–10 %**         |
