Overview

In this project, each team will apply multi-layer perceptrons (MLPs) to learn and approximate nonlinear relationships from data. By constructing, training, and evaluating neural networks, you will explore how MLPs can approximate complex relationships and how architecture, hyperparameters and optimization affect performance.

Tasks and Deliverables

# Task Description
1 Data loading and inspection Load the dataset, inspect its structure and dimensions, and briefly describe its content.
2 Data preprocessing Transform the data into a form suitable for MLP input:
– Convert to numpy / torch tensors
– Flatten or reshape inputs as needed
– Split into training and test sets
3 Model definition Define an appropriate MLP architecture (e.g with PyTorch nn.Sequential). Include number of layers, activation functions, and output dimensions. Report structure and number of parameters
4 Model training Implement a full training loop including:
– Loss function and optimizer
– Iterative training with loss updates
– Plot the loss history
5 Model evaluation Generate predictions on the test set and compute suitable metrics:
– Regression: MSE or R²
– Classification: overall accuracy and class-wise metrices
6 Visualization Plot training and validation loss curves, and visualize model results e.g. predicted vs. true values, or confusion matrices.
7 Discussion Provide a short reflection on model behavior and performance.

import numpy as np
import matplotlib.pyplot as plt
from sklearn.datasets import make_moons

def make_spirals(n_samples=1000, noise=0.2, n_classes=2, random_state=None):
    """
    This function creates n_samples x n_classes data points in 2D
    where each data points belongs to one of n_classes spirals (+ noise)
    Output: 
        - X: X.shape [n_samples,2]  (coordinates)
        - y: y.shape [n_samples,]    (class ID)
    """
    rng = np.random.default_rng(random_state)
    X = np.empty((n_samples * n_classes, 2), dtype=float)
    y = np.repeat(np.arange(n_classes), n_samples)

    r = np.linspace(0.0, 1.0, n_samples)

    for k in range(n_classes):
        t = np.linspace(4*k, 4*(k+1), n_samples) + rng.normal(scale=noise, size=n_samples)
        start = k * n_samples
        X[start:start+n_samples] = np.column_stack((r * np.sin(t), r * np.cos(t)))

    return X, y

n, nc = 1000, 3 
sig = 0.1
seed = 42

# choose one or both
X, y = make_moons(n_samples=n, noise=sig, random_state = seed) 
X, y = make_spirals(n_samples=n, n_classes=nc, noise=sig, random_state=seed)
