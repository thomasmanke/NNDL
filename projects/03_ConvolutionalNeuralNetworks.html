In this project, you will build and evaluate Convolutional Neural Network models (CNN) for image classification.

Datasets

The project-specific datasets and the target variable of interest are

Project ID Data Set Target Variable (\(y\))
3.1 WikiArt Artist (optionally: style/genre)
3.2 X-rays Normal/Pneumonia

Tasks

Task What to Do Key Points / Checks
Data Exploration Explore dataset origin, structure, dimensions, metadata; plot class distribution; check class balance;
Data Filtering Remove rare classes (<100 samples). Keep label indices consistent; Hint: keep mapping dictionaries.
Stratified Split Create train/val/test sets = 80/10/10 Use stratified sampling to preserve class distribution.
Image Transformations Define and apply data transformations + augmentation Resize; fix channels; normalize; augment; convert to tensors; Hint: inverse function tensor2image() may also be useful
Label Transformations Ensure consistent label encoding; reindex if needed. Check labels after filtering; update mappings.
Data Loaders Build train/val/test DataLoaders. Verify shapes, label formats, and data augmentation behavior.
Simple CNN Model build a CNN with at least 2 convolutional layers (incl. ReLU, MaxPool + optional dropout). Define input and output dimensions; ensure appropriate layer dimensions; report trainable parameter count.
Loss & Optimizer Define and explain a suitable loss function and optimizer Ensure compatibility with problem type and consider possible regularization
Model Training Train for sufficient number of epochs using mini-batches Plot loss curves for train + validation set. Discuss the observed behaviour (optionally: implement early stopping criteria)
Model Evaluation Evaluate trained model on test set Compute confusion matrix and report per-class metrics. Critique model and suggest possible improvements
Transfer Learning Load pretrained ResNet50; freeze all backbone parameters but replace final fully connected layer with suitable last layer Train as before and compare performance gain vs. simple CNN.
